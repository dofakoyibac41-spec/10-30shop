## Troubleshooting

### 🔴 `ModuleNotFoundError` при запуске
**Причина:** зависимости не установлены
**Решение:** `pip install -r requirements.txt` в папке `backend/`

### 🔴 `Connection refused` к PostgreSQL
**Причина:** БД не запущена или неверный `DATABASE_URL`
**Решение:** убедись что `docker compose up db -d` прошёл, проверь `.env`

### 🔴 `401 Unauthorized` при всех запросах
**Причина:** токен истёк или не передаётся
**Решение:** выйти и войти снова; убедись что cookie включены в браузере

### 🟡 `alembic upgrade head` падает
**Причина:** конфликт миграций или нет соединения с БД
**Решение:** проверь `DATABASE_URL` в `.env`, запусти `alembic history`

## 📋 СТАНДАРТ CHANGELOG.md

### Правила семантического версионирования (SemVer)

```
Формат: MAJOR.MINOR.PATCH

MAJOR — несовместимые изменения API, рефакторинг архитектуры
        Пример: 1.0.0 → 2.0.0 (сломали REST контракт)

MINOR — новая функциональность без поломки существующего
        Пример: 1.0.0 → 1.1.0 (добавили новый endpoint)

PATCH — исправления багов, патчи безопасности
        Пример: 1.0.0 → 1.0.1 (исправили баг в логине)

Суффиксы:
  1.0.0-alpha.1 — ранняя версия, нестабильная
  1.0.0-beta.1  — функционально готова, идёт тестирование
  1.0.0-rc.1    — release candidate, только критические баги
```

### Почему JWT в httpOnly cookie (а не localStorage)
[Объяснение: защита от XSS атак — из localStorage токен можно украсть через JS]

### Почему PostgreSQL (а не MongoDB)
[Объяснение: структурированные данные пользователей, транзакционность важна для биллинга]

## Поток аутентификации
```
1. POST /api/auth/login → сервер создаёт JWT
2. JWT устанавливается в httpOnly cookie
3. Все запросы автоматически отправляют cookie
4. Middleware проверяет JWT на каждом защищённом endpoint
```

## Что где лежит и почему
| Директория | Назначение |
|-----------|----------|
| `backend/app/routes/` | Только HTTP-логика, тонкий слой |
| `backend/app/services/` | Вся бизнес-логика |
| `backend/app/models/` | SQLAlchemy модели (таблицы) |
| `frontend/app/` | Next.js страницы (App Router) |
| `frontend/lib/api.ts` | Все запросы к бэкенду централизованно |
```
